# TestEraErase
テスト元号設定削除ツール
======================
Windows10 2018 Spring Release で入れられる(と思われる)新元号テスト用設定を削除するプログラムです。

言語・開発環境
------
C#/.NET Framework 4.0/VisualStudio2017

使い方
------
TestEraErase.exeをローカルドライブ上に置いてから実行する。

関連情報
------
１、レジストリを弄るので管理者権限で起動します  
２、レジストリを弄るので要注意  
　　実行結果は自己責任で。
　　万が一の為にシステム全体をバックアップしておいてください。  
　　不安な場合は使用しないでください。
